# LagosHomes

A modern real estate platform for browsing, searching, and viewing homes for sale in Lagos and Nigeria.

## Features

- Browse featured and all properties
- Search and filter by location, price, bedrooms, and property type
- View property details with image gallery
- Save favorite properties locally
- Contact agent form
- Responsive UI with Tailwind CSS
- Backend API with Express and MongoDB-ready configuration

## Setup

1. Install dependencies:

   ```bash
   npm install
   npm run install-all
   ```

2. Start both apps:

   ```bash
   npm run dev
   ```

3. Open the frontend:
   - `http://localhost:5173`

4. Backend API will run on:
   - `http://localhost:5000`

## Environment

Copy `backend/.env.example` to `backend/.env` and add your MongoDB connection string if available.
